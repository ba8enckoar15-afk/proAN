# 🧬 ProAn — Анализатор белков (MVP)

## Данные:
- 16,617 проверенных белков человека (UniProt Swiss-Prot)
- Длина: 50-800 аминокислот
- Очищено: верхний регистр, без дубликатов

## Запуск (2 минуты):
```bash
pip install -r requirements.txt
streamlit run proan_dashboard.py
```

Откроется браузер: **http://localhost:8501**

## Функции:
- 🔍 Поиск по ID белка
- 📊 Круговая диаграмма состава аминокислот
- 📈 График гидрофобности (Kyte-Doolittle)
- 📋 Метрики: длина, GRAVY, масса, pI
